# 4strodev dotfiles
Here you can clone and apply my dotfiles to your system.

### Quick access
- **.config files**
    - [neovim](./neovim/nvim/)
    - [alacritty](./alacritty/alacritty)
    - [zellij](./zellij/zellij)
- [**home directory files**](./home-files)
    - [zsh](./home-files/zsh)

## Fonts
I use [nerd fonts](https://www.nerdfonts.com) for my system 

## Wallpapers
Go to [wallhaven](https://wallhaven.cc) to get nice wallpapers.

## Dotfile linker
Use [dotger](https://github.com/4strodev/dotger).

Zsh still under migration from [stow](https://www.gnu.org/software/stow/manual/html_node/Resource-Files.html)
